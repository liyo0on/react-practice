## Deployment

To deploy this project run

1- react:

```bash
  npm i react
```
2- bootstrap:

```bash
  npm i bootstrap
```
3-  react-bootstrap:

```bash
  npm i react-bootstrap
```

4-  react-icons:

```bash
  npm i react-icons
```
5-  uuid:

```bash
  npm i uuid
```
